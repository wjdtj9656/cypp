## 만들게 된 동기
이 게임의 경우 파티 플레이나, 실시간 거래를 할 수 있습니다.
어떠한 유저와 파티플레이나, 거래를 하기전에 미리 그 유저에 대해 알 수 있다면 좋겠다는 생각이 들었습니다.
그리하여 유저에 대해서 익명으로 코멘트를 남길 수 있는 서비스를 개발했습니다.
